# Brudnopis

Udostępniam materiały z czasów nauki języka. Może się komuś przyda w jakimś stopniu. To nie jest projekt i nie wywołuje się go z pomocą stack. Zawiera dużo komentarzy, wyniki wywołań funkcji, algebraiczne typy danych i wiele innych. Omawia z funkcje z dokumentacji Semigroup, Monoid, Functor, Applicative, Alternative, Monad, MonadPlus, Foldable, Traversable.  

Przykłady w brudnopisie pochodzą z:
- własnych pomysłów
- książki "Haskell programming from first principles" - Christopher Allen, Julie Moronuki (osobiście uważam to za najlepszą książkę do nauki aktualnie)
- książki "Real World Haskell" - Bryan O'Sullivan wydawnictwa O'REILLY (zaawansowana książka uzupełniająca)
- strony http://learnyouahaskell.com
